# 🤝 Contributing to كِسرة

First off, thanks for wanting to contribute! 🎉

## 🐛 Found a Bug?

Open an [issue](https://github.com/Lord1Egypt/Kesra/issues/new?template=bug_report.md) with:
- Clear description
- Steps to reproduce
- Platform & device info
- Screenshots if possible

## ✨ Want a Feature?

Open a [feature request](https://github.com/Lord1Egypt/Kesra/issues/new?template=feature_request.md):
- What's missing?
- Why would it be awesome?
- Which world does it fit?

## 🎨 Want to Create Art/Sound?

1. Check the [issues](https://github.com/Lord1Egypt/Kesra/issues) for art needs
2. Or just surprise us! Assets go in `assets/`
3. PNG for textures, OGG for audio, TTF for fonts

## 💻 Want to Code?

### Getting Started
1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USER/Kesra.git`
3. Open with **Godot 4.3+**
4. Create a branch: `git checkout -b feature/your-idea`

### Code Style
- **GDScript** with snake_case
- One class per file
- Comment in **English** (code) + **Arabic** (intent)
- Use `@onready` for node references
- Signals at the top, then vars, then methods

### Commit Messages
```
type(world/area): short description

Examples:
✨(world:science): add astronomy level with star bricks
🐛(gameplay): fix ball passing through paddle at high speed
🎨(ui): add papyrus scroll menu background
```

### Pull Request Process
1. Update the **CHANGELOG.md**
2. Make sure CI passes
3. Request review from **@Lord1Egypt**
4. Wait for approval 🚀
